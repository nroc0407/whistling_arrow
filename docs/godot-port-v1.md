- 프로젝트 위치: `godot/`

## 구현 범위

이 마일스톤은 HTML 기준 구현의 지속형 화살 기믹을 Godot 프로젝트로 옮긴 첫 실행 가능 버전입니다.

- 하나의 화살이 `RETURN`, `FLOAT`, `ATTACK` 상태 사이에서 위치와 속도를 유지합니다.
- 공격은 최근접 생존 과녁을 자동 선택하고 제한된 각속도로 추적합니다.
- 귀환은 하단 홈을 추적하고 도착 거리에서 감속합니다.
- 과녁 3개, 홈 HP, 점수, 격추 수, 파티클, 궤적과 락온 표시를 포함합니다.
- `AudioStreamMicrophone`과 `AudioEffectCapture`로 2,048개 샘플을 수집합니다.
- 정규화 자기상관, RMS, 포물선 보간, 평활, 중립 구간과 2회 명령 확인을 구현합니다.
- 기준 Hz 슬라이더와 마이크 없이 상태를 검증하는 `1`·`2`·`3` 디버그 입력을 제공합니다.

Godot 버전에서는 HTML의 `MAX_ENEMIES = 3` 의도를 실제 활성 과녁 3개로 구현했습니다. HTML은 시작 및 교체 흐름상 활성 과녁이 주로 2개였으므로 이 부분은 의도적인 차이입니다. 높은 순음이 배음 주기로 잘못 잡히는 경우를 줄이기 위해 첫 번째 강한 자기상관 피크를 선택하는 보정도 추가했습니다.
